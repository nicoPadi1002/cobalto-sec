# Introduccion

Para empezar el proyecto, usamos VirtualBox para crear máquinas virtuales. Tambien descargamos las imágenes ISO de pfSense y de una distro ligera (Xubuntu) para simular los clientes de red.

Pueden descargar lo necesario desde:

-[VirtualBox](https://www.virtualbox.org/wiki/Downloads) -[pfSense](https://atxfiles.netgate.com/mirror/downloads/) -[Xubuntu](https://xubuntu.org/download/)

# Instalacion de las Maquinas Necesarias

Procedemos a instalar VirtualBox y las maquinas virtuales con las siguientes especificaciones:

## pfSense (nuestro firewall).

Tipo: BSD
Versión: FreeBSD (64-bit) (pfSense está basado en FreeBSD).
1024 MB (1 GB) de memoria RAM.
Disco VDI de 10GB.
Dos adaptadores de red:
*Adaptador 1: Conectado a "Adaptador puente" para que se conecta a nuestra red local.
*Adaptador 2: Conectado a "Red interna" para conectar con las otras máquinas virtuales.

## Xubuntu (nuestro cliente).

Tipo: Linux
Versión: Ubuntu (64-bit)
512MB de memoria RAM.
Disco VDI de 8GB.
Un adaptadores de red:
\*Adaptador 1: Conectado a "Red interna" para conectar con las otras máquinas virtuales.
